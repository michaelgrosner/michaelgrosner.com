CoffeeMol offers the ability to view and manipulate molecular structures in any modern, standards compliant browser -- including many mobile devices (multi-touch support in development). CoffeeMol was developed by myself on the side while developing a web interface for simulated structures I am generating in my grad school work. It is written out of frustration stemming from having to load PDB files into PyMol, orient to a view, save the PNG file, load it onto the webserver, and remember the URL, all while only getting a few views of the molecule. Instead, with only a few lines of JavaScript can load an viewer capable of displaying complex molecules. In addition, it doesn't require antiquated browser applets. ### To embed into an existing webpage This will reproduce the viewer above, except the viewer above only contains the first PDB file in `structures` and the styling via CSS changes is different. * Compile -- You'll need to install [CoffeeScript](http://coffeescript.org/#installation), or copy the compiled JS [here](http://www.michaelgrosner.com/CoffeeMol.js) {% syntax bash %} $ coffee -cj CoffeeMol.coffee CanvasContext.coffee Element.coffee Structure.coffee \ Chain.coffee Residue.coffee Atom.coffee main.coffee Viewer.coffee {% endsyntax %} * Create a `` element with id `coffeemolCanvas`, include at least jQuery version 1.7.1, and the compiled `CoffeeMol.js` file. {% syntax html %} Canvas Load Failed ... {% endsyntax %} * Include the following JavaScript: {% syntax javascript %} // Change this dictionary to suit your PDB needs structures = { "http://www.rcsb.org/pdb/files/1MBO.pdb": { // URL to a well-behaved PDB file drawMethod: "both", // Use 'lines', 'points', or 'both' drawColor: [255, 0, 0] // RGB color (optional) }, "/static/pdbs/1KX5.pdb": { // Can mix structures and properties drawMethod: "points" borderColor: [100, 0, 0] } }; // CoffeeMol only exposes the CanvasContext object to `window` and a loader function ctx = window.ctx; window.loadFromDict(structures); // Once everything is loaded, run it ctx.init(); {% endsyntax %} ### To use the full, interactive mode Go to the [CoffeeMol Interactive Viewer](http://www.michaelgrosner.com/coffeemol) And enjoy!
