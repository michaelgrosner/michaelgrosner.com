It's like Mathematica, but runs Python underneath. Probably the most mature open-source and free Python compatible cheminformatics software is [OpenBabel](http://ipython.org/ipython-doc/dev/interactive/htmlnotebook.html), which supports SVG drawing of molecular structures from SMILES or nearly any other format.
